[로그인]
- 아무것도 입력하지 않고 로그인 눌렀을 때 > 남은 창 입력 메세지
- 아이디만 입력했을 때 > 남은 창 입력 메세지
- 비밀번호만 입력했을 때 > 남은 창 입력 메세지
- 잘못된 아이디나 비밀번호를 입력했을 때 > 아이디 혹은 비번 확인 메세지
- 모두 올바르게 잘 입력했을 때 > 로그인 됨
- admin 테이블에 아이디와 해쉬함수화된 비밀번호 입력되어있는지

[대시보드]
- 총 회원 수 / 진행중인 수업 / 이번달 매출 / 오늘 스케줄 모두 잘 뜨는지

[강사 관리]
1) 강사 등록 시
    - 이름 / 월급 중 하나 빼고 등록했을 시 > 남은 창 입력 메세지
    - 전화번호 입력 안하고 등록했을 시 > 제대로 등록 되고 전화번호란은 '-' 표기
    - 등록 시 instructors 테이블에 제대로 들어가는지
2) 강사 수정 시
    - 이름 / 월급 중 하나 빼고 수정했을 시 > 남은 창 입력 메세지
    - 전화번호 입력 안하고 등록했을 시 > 제대로 등록 되고 전화번호란은 '-' 표기
    -수정 시 instructors 테이블에 제대로 들어가는지
3) 강사 삭제 시
    : 강사 목록에서 제거 후 instructors 테이블에서도 제대로 삭제되어야함

[수업 관리]
1) 수업 등록 시
    - 주간 수업 횟수 조정 후 시간 선택했을 시 > 시간 선택란이 조정한 횟수만금 조정되어야함
    - 주간 수업 횟수와 맞지 않는 개수의 시간대를 선택했을 시 > 맞는 개수만큼 선택해달라는 메세지 떠야함
    - 겹치는 시간대 선택했을 시 > 시간 선택 모달에서 확인 눌렀을 때 겹치는 시간대 있다는 메세지 떠야함
    - 시간, 프로그램명, 프로그램 가격 중 하나라도 빠뜨렸을 시 > 3개 모두 입력 바란다는 메세지 떠야함
    - 강사, 특이사항 입력 안할 시 > '-'로 표기되어야함
    - 프로그램 색 선택 시 > 보이는 색상으로 바뀌어야 함
    - 오류 없을 시 > programs 테이블 및 class_schedules 테이블에 제대로 등록되어야함
2) 수업 수정 시
    - 프로그램에 화살표 가져다 댈 시 > 약간 커져야함
    - 프로그램 클릭했을 시 > 제대로 데이터 채워진 수정 모달이 열려야 함
    - 주간 수업 횟수 조정 후 시간 선택했을 때 > 시간 선택란이 조정한 횟수만금 조정되어야함
    - 주간 수업 횟수와 맞지 않는 개수의 시간대를 선택했을 시 > 맞는 개수만큼 선택해달라는 메세지 떠야함
    - 겹치는 시간대 선택했을 시 > 시간 선택 모달에서 확인 눌렀을 때 겹치는 시간대 있다는 메세지 떠야함
    - 시간, 프로그램명, 프로그램 가격 중 하나라도 빠뜨렸을 시 > 3개 모두 입력 바란다는 메세지 떠야함
    - 강사, 특이사항 입력 안할 시 > '-'로 표기되어야함
    - 프로그램 색 선택 시 > 보이는 색상으로 바뀌어야 함
    - 오류 없을 시 > programs 테이블 및 class_schedules 테이블에 제대로 수정되어야함
3) 수업 삭제 시
    - 2개 이상 시간대의 수업인 경우 하나를 클릭해도 나머지 모두 삭제돼야 함
    - 수업 삭제 버튼을 다시 누르지 않는 이상 연속해서 수업을 삭제할 수 있어야 함.
    - 오류 없을 시 > programs 테이블 및 class_schedules 테이블에서 제거되어야함.
4) 수업 초기화시
    - 잘못된 비밀번호 입력 시 > 초기화 불가 메세지 떠야함
    - 올바른 비밀번호 입력 시 > 제대로 초기화되어 programs 테이블 및 class_schedules에서 모든 데이터가 제거되어야함
